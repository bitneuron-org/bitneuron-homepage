# BitNeuron Homepage

An open source homepage for BitNeuron.

## Available Scripts

Install the dependencies.
```
yarn install
```

Start the webapp locally
```
yarn start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

Builds the app for production.
```
yarn build
```

## Deployment

```
./scripts/deploy.sh
```
